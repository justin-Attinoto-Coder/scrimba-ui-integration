# Scenic Forests - Modern Responsive Website

A state-of-the-art, responsive website for Scenic Forests, showcasing luxury cabin rentals in a breathtaking forest setting. This project demonstrates modern web development practices with interactive elements, seasonal gradient themes, and unified visual design.

## 🌟 Features

- **Interactive Cabin Modals**: Detailed cabin information with smooth GSAP animations and backdrop blur effects
- **Seasonal Gradient Themes**: Dynamic color schemes for Spring, Summer, Autumn, and Winter sections
- **Unified Forest Imagery**: Consistent forest background across all pages for immersive user experience
- **Responsive Design**: Mobile-first approach with Tailwind CSS ensuring perfect display on all devices
- **Consistent Navigation**: Unified header and footer across all pages with smooth transitions
- **Modern UI Elements**: Gradient overlays, backdrop blur, and sophisticated visual effects
- **Accessibility**: Semantic HTML5 structure and keyboard navigation support

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| [Vite 6](https://vitejs.dev) | Lightning-fast dev server & bundler for modern web development |
| [Tailwind CSS 4](https://tailwindcss.com) | Utility-first CSS framework with advanced gradients and responsive utilities |
| [GSAP 3.12.2](https://greensock.com/gsap/) | High-performance animation library for smooth modal transitions |
| JavaScript ES6+ | Event handling, dynamic content population, and interactive functionality |
| HTML5 | Semantic structure with modern CSS techniques and accessibility features |

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/justin-Attinoto-Coder/scrimba-ui-integration.git
cd scrimba-ui-integration
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The website will be available at `http://localhost:5173/`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 🌐 Deployment to Vercel

This project is optimized for deployment on Vercel:

1. **Connect Repository**: Link your GitHub repository to Vercel
2. **Auto-Deploy**: Vercel will automatically build and deploy on every push to main
3. **Custom Domain**: Easily add a custom domain through Vercel's dashboard
4. **Environment Variables**: Configure any environment variables if needed (none required for this project)

The build command is already configured in `package.json` and `vite.config.js` for seamless deployment.

## 🎨 Design System & Pattern Library

This project includes a comprehensive design system that demonstrates modern web development practices:

### Key Design System Features:
- **Seasonal Color Themes**: Spring (green/emerald), Summer (lime/emerald), Autumn (olive/teal), Winter (blue/slate)
- **Component Library**: Reusable UI components including buttons, cards, navigation, and modals
- **Typography Scale**: Consistent text hierarchy using Montserrat font family
- **Layout Patterns**: Responsive grid systems and spacing guidelines
- **Accessibility**: AAA contrast ratios and keyboard navigation support

### Dark Mode Implementation:
- **Default Dark Theme**: Modern dark mode as the default experience
- **Theme Toggle**: Sun/moon icon toggle in the upper right corner
- **Persistent Preferences**: Theme choice saved in localStorage
- **Smooth Transitions**: CSS transitions for seamless theme switching

### Pattern Library Contents:
- Color system with seasonal palettes
- Typography guidelines and usage
- Component examples with code snippets
- Layout patterns and spacing rules
- Usage guidelines for each theme
- Implementation notes and best practices

Access the live design system at: `design-system.html`
